# 模板設計說明
## 結構
- `主大小`, `中心區塊`(`3x3` 區塊的中心 `1` 格區塊): `16`x`15`x`16`
- `分支大小`, `延伸區塊`(`3x3` 區塊的外圍 `8` 格區塊): `16`x`15`x`16`
- `材質選用`:
    - `地板`: {無須地板, 地板與下一層天花板結合}
    - `牆壁`: {
        [剝皮竹方塊:`minecraft:stripped_bamboo_block`],
        [剝皮橡木塊:`minecraftstripped_oak_wood`]
    }
    - `天花板`: {
        [浮雕書櫃:`minecraft:chiseled_bookshelf`],
        [黃色蛙光體:`minecraft:ochre_froglight`],
        [沙子:`minecraft:sand`]
    }
    - `裝飾`: {無限制}

---

## 注意
- 模板結構長寬限制:
    - 上限: `3x3` 區塊
    - 低標: `1x1` 區塊
- 關於結構:
    - 結構中心 `1x1` 區塊為主結構, 會保留較完整的結構內容
    - 結構外圍 `8` 格區塊為分支結構, 會與其他結構進行交叉生成, 不會保留完整結構內容
- 天花板的上方一層固定鋪沙子補滿剩餘空間(上一層的地板)
- 裝飾適量不要太多, 你也不想看到後室裡全是裝飾對吧
- 無須蓋地板(除非要加高結構整體)
- 結構空間:
    - 上限:  `48` 格寬(`3x3` 區塊的上限)
    - 建議: `5` 格寬
    - 低標: `1` 格寬 (但不要太過平凡的出現)
